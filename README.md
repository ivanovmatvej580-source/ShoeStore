\# 👟 StyleStep - Магазин обуви



Приложение для управления магазином обуви на C# (.NET 8) с использованием Windows Forms и Web API.



Запустите API сервер (окно 1) PowerShell

cd ShoeStore.api

dotnet restore

dotnet run

Запустите клиентское приложение (окно 2) PowerShell

cd ShoeStore

dotnet restore

dotnet run
